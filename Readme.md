# Requirements:

感谢观看，希望你喜欢这个视频。接下来我来详细介绍一下规则。

游戏步骤：

>+ 1、分发身份牌并摇骰子，秘密查看身份及骰子点数。

>+ 2、天黑请闭眼，主持人叫到点数时，对应点数的玩家睁眼并行动。
如果一个嗜睡老鼠独自醒来，可以秘密地查看任意一名玩家的骰子。
如果不止一个人醒来，则不能查看骰子。
无论多少人睁眼，大盗醒来都一定要偷走奶酪。

>+ 3、天亮前，所有人伸出手，大盗睁眼选择共犯，共犯睁眼与大盗确认。

>+ 4、公开讨论后，投票指认奶酪大盗。
嗜睡老鼠胜利条件：正确指认奶酪大盗
大盗及共犯胜利条件：嗜睡老鼠或共犯被指认。
也就是说，共犯需要充当挡箭牌。
